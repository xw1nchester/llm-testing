- subject - (субъект Российской федерации)
- year (год, число)
- value (число на пересечении региона и года)

Данные брать только из столбца **Все население, мужчины и женщины**.

Сохраняй названия субъектов и городов точно как в исходном CSV, без изменений/сокращений, без добавления/удаления пробелов.

Выведи только валидный JSON (полностью), без объяснений.
