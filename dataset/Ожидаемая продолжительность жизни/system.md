Твоя задача - преобразовать входные данные в формате CSV в формат JSON. В результате должен получиться массив объектов с полями:
- subject - Тюменская область (кроме Ханты-Мансийского автономного округа-Югры и Ямало-Ненецкого автономного округа)
- year (год, число)
- value (число на пересечении региона и года)

Данные брать только из строки **Оба пола, все население**.

Выведи только валидный JSON (полностью), без объяснений.
